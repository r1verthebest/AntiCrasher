# AntiCrasher
Solução avançada de segurança para Minecraft 1.8.9. Atua na camada Netty interceptando pacotes brutos para mitigar ataques de Bot (DoS), exploits de NBT e Book Crash. Focado em performance e baixo nível, filtra dados antes da main thread para garantir estabilidade e proteção total contra pacotes maliciosos.
